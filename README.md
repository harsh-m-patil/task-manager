# Task Manager

Task Management Dashboard monorepo (Next.js + Turborepo + shadcn/ui).

Deployed link -> [tasks](https://tasks.harshmpatil.xyz/)

## Run the app

### Install

```bash
pnpm install
```

### Start development

```bash
pnpm dev
```

Open `http://localhost:3000`.

### Build

```bash
pnpm build
```

### Preview production build

```bash
pnpm --filter web start
```
